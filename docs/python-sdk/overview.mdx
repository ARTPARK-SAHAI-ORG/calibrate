The Pense Python SDK provides programmatic access to all evaluation and simulation features.

## Installation

Install using pip:

```bash
pip install pense
```

Or using uv:

```bash
uv add pense
```

## Configuration

Create a `.env` file in your project root and add the API keys for the providers you want to use. All modules (agent, stt, tts, llm) will use this single `.env` file.

```bash
# Speech to Text providers
DEEPGRAM_API_KEY=your_key
OPENAI_API_KEY=your_key
GOOGLE_APPLICATION_CREDENTIALS=/path/to/credentials.json
SARVAM_API_KEY=your_key
ELEVENLABS_API_KEY=your_key
CARTESIA_API_KEY=your_key
GROQ_API_KEY=your_key

# Text to Text providers
OPENROUTER_API_KEY=your_key
```
