The Pense Python SDK provides programmatic access to all evaluation and simulation features.

## Installation

```bash
git clone https://github.com/artpark/pense.git
cd pense
uv sync --frozen
```

## Modules

| Module | Description |
|--------|-------------|
| `pense.stt` | Speech-to-Text evaluation |
| `pense.tts` | Text-to-Speech evaluation |
| `pense.llm` | LLM tests and simulations |
| `pense.agent` | Voice agent simulations |
