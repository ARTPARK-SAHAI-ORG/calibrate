The Calibrate Python SDK provides programmatic access to evaluate Speech to Text, Text to Speech, LLMs, and run full voice agent simulations.

## Installation

```bash
pip install calibrate
```

## Setup

Create a `.env` file with API keys for providers you plan to use:

```bash
# Speech to Text / Text to Speech
DEEPGRAM_API_KEY=your_key
OPENAI_API_KEY=your_key
GOOGLE_APPLICATION_CREDENTIALS=/path/to/credentials.json
SARVAM_API_KEY=your_key
ELEVENLABS_API_KEY=your_key
CARTESIA_API_KEY=your_key
GROQ_API_KEY=your_key
SMALLEST_API_KEY=your_key

# LLM
OPENROUTER_API_KEY=your_key
```
