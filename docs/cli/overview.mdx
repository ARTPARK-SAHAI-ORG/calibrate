## Installation

```bash
pip install calibrate-agent
```

## Get started

<Note>
  The interactive CLI requires [Node.js](https://nodejs.org/) to be installed on
  your system.
</Note>

```bash
calibrate
```

Launches the interactive main menu where you can select from all available options.

| Command                 | Description                                       |
| ----------------------- | ------------------------------------------------- |
| `calibrate stt`         | Interactive STT evaluation — benchmark providers  |
| `calibrate tts`         | Interactive TTS evaluation — benchmark providers  |
| `calibrate llm`         | Interactive LLM evaluation — test agent responses |
| `calibrate simulations` | Interactive text or voice simulations             |
