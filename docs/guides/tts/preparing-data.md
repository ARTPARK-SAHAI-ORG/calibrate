# Preparing TTS Data

Set up your test data for TTS evaluation.

## CSV Format

Create a CSV file with a single `text` column:

```csv
text
hello world
this is a test
```

## Example File

Save as `tts-input.csv`:

```csv
text
Hello, how are you today?
The weather is beautiful today.
```
